# Flipbook (GitHub Pages)

1. 이 폴더(`flipbook/`) 내용을 GitHub 리포지토리 루트에 업로드하세요.
2. GitHub → Settings → Pages → Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)` 설정.
3. 배포 주소로 접속하면 `ebooks/manifest.json`에 적힌 PDF를 드롭다운에서 선택해 볼 수 있습니다.
   - 직접 링크 열기: `?file=./ebooks/무료전자책2025.pdf`
4. 로고 바꾸기: `assets/logo.png` 교체.
5. 색상 바꾸기: `index.html` 상단 `:root` CSS 변수 수정.
