# Flipbook (Local PDF.js version)

1) /libs/ 폴더에 아래 두 파일을 넣으세요 (CDN 불필요):
   - pdf.min.js
   - pdf.worker.min.js
   다운로드: https://github.com/mozilla/pdfjs-dist/tree/master/build

2) GitHub Pages 배포:
   - Settings → Pages → Source: Deploy from a branch
   - Branch: main / Folder: / (root)

3) 사이트 주소 예: https://<아이디>.github.io/flipbook/
